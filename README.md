# Vecta Bio — Link na Bio

Site desenvolvido para a **Vecta Soluções Contábeis**, uma empresa real de contabilidade focada em MEI e pequenos empreendedores.

## 📌 Sobre o projeto

Link na bio criado para o Instagram da Vecta, reunindo em uma única página os principais canais de atendimento e informações da empresa.

## 🚀 Funcionalidades

- Contato direto via WhatsApp com os sócios
- Link para o Instagram
- Seção "Por que a Vecta?" com os diferenciais da empresa
- Design responsivo mobile first
- Animações de entrada

## 🛠️ Tecnologias

- React
- Vite
- CSS puro (variáveis, flexbox, grid, animações)
- Google Fonts (Bebas Neue + DM Sans)
- Deploy na Vercel

## 🎨 Identidade Visual

Baseada na identidade da Vecta — azul escuro, ciano e branco.
